# Bubble Pop Quest

Unity 2022.3 LTS 2D Android bubble shooter scaffold.

## Open and Build

1. Open this folder in Unity Hub with Unity 2022.3 LTS.
2. Let Unity restore packages from `Packages/manifest.json`.
3. Run `Bubble Shooter > Prepare Android Build`.
4. Open `Assets/Scenes/Splash.unity` and press Play.
5. Configure live LevelPlay app IDs and Google Play product IDs before release builds.

The current monetization layer uses safe stubs so gameplay, rewarded choices, and store grant flows can be tested offline.
Unity will generate local `ProjectSettings` defaults on first open; the menu command applies the Android portrait settings afterward.
